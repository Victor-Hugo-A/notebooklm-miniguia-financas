# Miniguia NotebookLM — Educação financeira básica

Este repositório documenta a criação de um caderno temático no NotebookLM com **5 fontes abertas** sobre:
**orçamento**, **juros compostos** e **inflação/IPCA**.

O projeto inclui:
- objetivos de estudo
- perguntas estratégicas
- prompts testados (com variações)
- registro de respostas com referências às fontes
- miniguia final e glossário

## Estrutura do repositório
- `01-fontes/fontes.md` — lista das 5 fontes (links + justificativa)
- `02-objetivos/objetivos.md` — objetivos de estudo
- `03-perguntas/perguntas.md` — perguntas estratégicas
- `04-prompts/prompts.md` — prompts reutilizáveis + variações
- `05-registros/respostas-com-referencias.md` — respostas do NotebookLM com referências
- `06-miniguia/miniguia.md` — miniguia final (resumo + comparação + checklist)
- `06-miniguia/glossario.md` — glossário de conceitos

## Como reproduzir no NotebookLM (resumo)
1. Criar um notebook com o tema.
2. Adicionar as 5 fontes listadas em `01-fontes/fontes.md`.
3. Executar os prompts em `04-prompts/prompts.md`.
4. Registrar as respostas e referências em `05-registros/`.
5. Consolidar o material em `06-miniguia/`.
